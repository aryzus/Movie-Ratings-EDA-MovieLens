# EDA-PROJECT

🎬 Movie Ratings EDA (MovieLens Dataset)


Overview

This project performs Exploratory Data Analysis (EDA) on the MovieLens dataset to uncover patterns in user ratings, movie popularity, and genre preferences.



Dataset
Source: MovieLens
~100,000 ratings
Includes users, movies, ratings, and timestamps


Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn


Key Insights
Average rating is ~3.5, indicating positive user bias
Popular movies have more stable ratings
Classic movies dominate rating counts
Genres like War and Documentary score higher
User activity has increased over time

Visualizations
Rating distribution
Top rated movies
Popularity vs rating scatter plot
Genre analysis
User activity trends

Outcome

This project demonstrates how EDA can uncover meaningful insights from user-generated data and supports building recommendation systems.
